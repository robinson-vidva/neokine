# Sample media

Bundled demo assets so the web app has something to run without your own file.
They load via the "Try a sample" buttons through the same client-side pipeline
as an uploaded file — nothing is sent anywhere.

All samples are **AI-generated** with Google's models (Veo / Gemini) and depict
**fully synthetic people — not real, identifiable individuals, and not
infants.** Because they are machine-generated and show no real person, there are
no third-party copyright, model-release, privacy, or consent constraints on
bundling them here. This is consistent with the app's scope: a general
pose-visualization demo, not an infant feed and not a clinical measurement.

## Files

- **synthetic-walk.mp4** — short clip of a synthetic person moving (used by the
  video sample button).
- **synthetic-pose.jpg** — a single full-body frame of a synthetic person (used
  by the image sample button).
