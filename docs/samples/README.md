# Sample media

Bundled demo clips so the web app has something to run without your own file.
They load via the "Try a sample" button through the same client-side pipeline as
an uploaded file — nothing is sent anywhere.

## synthetic-walk.mp4

- **AI-generated** with Google's video model (Veo / Gemini). It depicts a
  **fully synthetic person — not a real, identifiable individual, and not an
  infant.**
- Used as demonstration content only. Because it is machine-generated and shows
  no real person, there are no third-party copyright, model-release, privacy, or
  consent constraints on bundling it here.
- Consistent with the app's scope: a general pose-visualization demo, not an
  infant feed and not a clinical measurement.
